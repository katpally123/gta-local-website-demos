# DSL Mechanical Website

Production static website package for DSL Mechanical, a Kitchener-Waterloo plumbing and mechanical services business.

## Files

- `index.html` - responsive one-page business website with SEO metadata and schema markup
- `styles.css` - custom visual system and responsive layout
- `research.md` - public research notes and source list
- `outreach-email.md` - Gmail outreach draft copy
- `deployment-status.md` - deployment and verification notes
