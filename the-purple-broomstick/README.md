# The Purple Broomstick Website

Production static website for The Purple Broomstick in Cornwall, Ontario.

## Files

- `index.html` - finished page with SEO metadata and local business schema
- `styles.css` - responsive custom brand styling
- `script.js` - mobile menu, FAQ toggles, and reveal animation
- `research.md` - public research notes and source links
- `outreach-email.md` - Gmail draft body
- `deployment-status.md` - deployment verification notes
